ИНСТРУКЦИЯ ПО ЛАЙФ-РЕЛОАД ДЛЯ ВИНДЫ

browser-sync start --server --files **/*.*

 Ребята, инструкция как сделать лайв-релоад вашей верстки. (для пользователей Windows, у кого мак/линукс - пишите, сделаю отдельно)


1) скачиваем и устанавливаем https://git-scm.com (на всякий случай, видео: https://www.youtube.com/watch?v=rWboGsc6CqI)


2) скачиваем и устанавливаем https://nodejs.org/en/


3) открываем программу Git Bash (то, что вы установили первым). Не пугайтесь, это консоль, вам теперь с ней работать частенько придется))


4) проверяем, что нода установилась, в консоли вводим `node -v`


5) если все ок вводим `npm install -g browser-sync`

Что мы только что сделали:

Установили нормальную консоль на винду :slightly_smiling_face:

Поставили ноду - это окружение для исполнения javascript кода

командой `npm install` поставили пакет browser-sync, который делает магию

Теперь:

заходим в папку проекта (там, где лежит index.html)

кликаем по папке правой кнопкой мыши и выбираем git bash here

в консоли вводим команду `browser-sync start --server --files **/*.*` - вуаля, ваша верстка у вас в браузере, и обновляется при сохранении исходников)
